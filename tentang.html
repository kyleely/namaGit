html adalah bahasa markup yang digunakan untuk membuat halaman web. HTML merupakan singkatan dari HyperText Markup Language. HTML digunakan untuk menyusun dan mengatur konten di dalam sebuah halaman web, seperti teks, gambar, tautan, dan elemen lainnya. Dengan menggunakan HTML, pengembang web dapat membuat struktur dan tampilan halaman web sesuai dengan kebutuhan mereka. HTML juga memungkinkan pengguna untuk menambahkan elemen interaktif seperti formulir, tombol, dan multimedia ke dalam halaman web mereka.
